# Good Cup — Digital Calling Card

A self-contained digital business card for **Xandro Ysagani Zarate** — Head Roaster & R&D, Good Cup Coffee Co. (Cebu, Philippines).

- Single, self-contained [`index.html`](index.html) — fonts, logo, and vCard are all embedded, so it makes **no external network requests**.
- Tap **Save Contact** to download a vCard (`.vcf`).
- Built on the Good Cup brand system: Crema / Robusta palette and the Good Sans typeface.

## Usage

Open `index.html` in any browser, or drop it on any static host.
